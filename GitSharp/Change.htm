Public properties

AbstractObject ChangedObject get;

Always returns an object, no matter what kind of change. It normally returns the ComparedCommit's version of the changed object except for ChangeType.Removed where it returns the ReferenceCommit's version of the object. This property is designed to release the calling code from null checking and version selection and may be especially useful for GUI bindings.
